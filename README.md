# Туристический портал

Веб-приложение на Django для просмотра туристических направлений, достопримечательностей и отзывов.

## Возможности

- Просмотр стран, городов и достопримечательностей
- Система рейтингов и отзывов
- Галерея фотографий для каждой достопримечательности
- Регистрация и авторизация пользователей
- Административная панель для управления контентом
- Адаптивный дизайн

## Требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

## Установка и запуск

### Шаг 1: Скачайте проект

Скачайте все файлы проекта в одну папку на вашем компьютере.

### Шаг 2: Откройте командную строку

**Windows:**
- Нажмите Win + R
- Введите `cmd` и нажмите Enter
- Перейдите в папку с проектом: `cd путь\к\папке\проекта`

Пример:
```
cd C:\Users\Андрей\Downloads\tourism_portal
```

### Шаг 3: Установите зависимости

```
python -m pip install -r requirements.txt
```

Подождите, пока установятся все библиотеки.

### Шаг 4: Создайте базу данных

```
python manage.py makemigrations
python manage.py migrate
```

### Шаг 5: Создайте администратора

```
python create_superuser.py
```

Будет создан администратор с данными:
- Логин: `admin`
- Пароль: `admin123`

### Шаг 6: Добавьте тестовые данные

```
python add_sample_data.py
python add_more_data.py
```

Эти команды добавят в базу страны, города, достопримечательности и отзывы.

### Шаг 7: Запустите сервер

```
python manage.py runserver
```

Вы увидите сообщение:
```
Starting development server at http://127.0.0.1:8000/
```

### Шаг 8: Откройте сайт

Откройте браузер (Chrome, Firefox, Edge) и перейдите по адресу:

**http://127.0.0.1:8000/**

## Использование

### Главная страница

http://127.0.0.1:8000/

Здесь отображаются все доступные страны. Кликните на любую страну, чтобы увидеть её города и достопримечательности.

### Регистрация и вход

- Регистрация: http://127.0.0.1:8000/accounts/register/
- Вход: http://127.0.0.1:8000/accounts/login/
- Профиль: http://127.0.0.1:8000/accounts/profile/

После входа вы сможете оставлять отзывы о достопримечательностях.

### Административная панель

http://127.0.0.1:8000/admin/

Логин: `admin`
Пароль: `admin123`

В админ-панели вы можете:
- Добавлять новые страны, города и достопримечательности
- Загружать фотографии (главное фото + галерея)
- Управлять отзывами пользователей
- Редактировать существующие записи

## Как добавить фотографии к достопримечательности

1. Зайдите в админ-панель: http://127.0.0.1:8000/admin/
2. Войдите (admin / admin123)
3. Нажмите на "Достопримечательности"
4. Выберите нужную достопримечательность или создайте новую
5. Прокрутите страницу вниз до раздела "Фотографии достопримечательностей"
6. Нажмите "Добавить еще одну Фотография достопримечательности"
7. Выберите файл изображения с вашего компьютера
8. Добавьте подпись к фото (необязательно)
9. Нажмите "Сохранить"

Фотографии появятся в галерее на странице достопримечательности.

## Остановка сервера

Чтобы остановить сервер, нажмите в командной строке:
- **Windows:** `Ctrl + C` или `Ctrl + Break`

## Структура проекта

```
tourism_portal/          - настройки Django проекта
destinations/            - приложение с моделями достопримечательностей
accounts/                - приложение авторизации
templates/               - HTML шаблоны
static/                  - статические файлы (CSS, JS)
media/                   - загруженные изображения
db.sqlite3              - база данных SQLite
manage.py               - утилита управления Django
```

## База данных

Проект использует SQLite - файловую базу данных (db.sqlite3). Она создается автоматически и не требует установки дополнительного ПО.

## Возможные проблемы

**Ошибка "python не является внутренней командой":**
- Убедитесь, что Python установлен
- Проверьте командой: `python --version`
- Если не работает, попробуйте: `python3 --version`

**Ошибка при установке Pillow:**
- Pillow требует компилятор C
- Скачайте готовый wheel-файл с https://pypi.org/project/Pillow/

**Порт 8000 занят:**
- Запустите на другом порту: `python manage.py runserver 8080`
- Откройте: http://127.0.0.1:8080/
